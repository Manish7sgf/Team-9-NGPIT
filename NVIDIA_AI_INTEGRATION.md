# NVIDIA AI Integration Guide

## 🚀 Using NVIDIA AI for Personalized Learning Paths

The Skill Gap Radar now includes **AI-powered personalized learning path recommendations** using the NVIDIA API (Llama 2 70B Chat).

---

## 🔐 Setup: Getting Your NVIDIA API Key

### Step 1: Create NVIDIA Account
1. Go to **https://build.nvidia.com/**
2. Sign up or log in with your account
3. Go to **API Keys** section
4. Click **Generate New API Key**
5. Copy the key (starts with `nvapi-`)

### Step 2: Store Securely in Your Project

**IMPORTANT: Never commit API keys to git!**

#### Option A: Environment Variables (Recommended)
Create a `.env` file in your project root:

```bash
# .env (DO NOT COMMIT THIS FILE)
VITE_API_KEY=nvapi-YOUR_KEY_HERE
VITE_ENABLE_AI_RECOMMENDATIONS=true
```

Add to `.gitignore`:
```
.env
.env.local
.env.*.local
```

#### Option B: localStorage (For Browser Use)
Store the key in browser after user input:

```javascript
// Save API key to localStorage
localStorage.setItem('nvidia_api_key', 'nvapi-YOUR_KEY_HERE');

// The analyzer will auto-detect it
```

#### Option C: Backend Proxy (Most Secure)
Never expose API keys in frontend code. Use a backend endpoint:

```javascript
// Frontend calls your backend
const response = await fetch('/api/analyze', {
  method: 'POST',
  body: JSON.stringify({ userSkills })
});
```

```javascript
// Backend (Node.js/Express)
app.post('/api/analyze', async (req, res) => {
  const apiKey = process.env.NVIDIA_API_KEY;
  const result = await callNvidiaAPI(req.body.userSkills, apiKey);
  res.json(result);
});
```

## 🔄 How It Works

### Analysis Flow

```
User enters skills
        ↓
Click "Analyse Gap →"
        ↓
1. Standard gap analysis (matching, missing, recommendations)
        ↓
2. If enableAI=true, calls NVIDIA API with:
   - Current skills
   - Missing skills
   - Skill gap percentage
        ↓
3. NVIDIA Llama 2 generates 3 personalized learning paths
        ↓
4. Results displayed in AILearningPathCard component
        ↓
onAnalysisComplete callback fires with full results
```

### API Call Details

**Endpoint:** `https://integrate.api.nvidia.com/v1/chat/completions`

**Model:** `nvidia/llama-2-70b-chat`

**Prompt Template:**
```
You are a career advisor specializing in 2030 workforce skills.

User Profile:
- Current Skills: Java, Python
- Missing Critical Skills: AI Collaboration, Prompt Engineering
- Skill Gap: 71%

Generate 3 personalized learning path recommendations in this format:
1. [Skill Name]: [2-3 sentence explanation] | [Time estimate]
2. ...
3. ...
```

**Response Parsing:** Extracts numbered items and parses into structured paths

## 🛡️ Error Handling

If the API fails, the system automatically falls back to **default learning paths**:

```javascript
// Fallback if API unavailable
generateFallbackPaths(analysisData)
// Returns time-tested learning paths without AI
```

This ensures the feature **always works**, even if:
- API key is missing
- Network is down
- API quota exceeded
- API service unavailable

## 🎨 Customization

### Custom Prompt for AI

Modify the prompt template in `skillGapAnalyzer.js`:

```javascript
const buildPrompt = (analysisData) => {
  const { matchedSkills, missingSkills, gapScore } = analysisData;
  
  return `[Your custom prompt here with analysis data]`;
};
```

Example: Add industry focus

```javascript
return `You are a career advisor for ${industry} roles.

User Skills: ${matchedSkills.map(s => s.skill).join(', ')}
Missing: ${missingSkills.map(s => s.skill).join(', ')}

Generate 3 learning paths that prioritize ${industry} relevance.`;
```

### Custom Learning Path Display

Modify `AILearningPathCard.jsx` to display paths differently:

```jsx
{learningPaths.learningPaths.map((path, index) => (
  <CustomPathComponent 
    key={path.id}
    path={path}
    number={index + 1}
  />
))}
```

### Disable AI Fallback

Force fallback paths always:

```jsx
<SkillGapRadar
  enableAI={false}  // Always use fallback paths
  userSkills={userSkills}
/>
```

## 🔧 Troubleshooting

### No API Key Error

**Problem:** "No NVIDIA API key found. Using fallback recommendations."

**Solution:**
1. Create `.env` file with `VITE_API_KEY=nvapi-YOUR_KEY`
2. Or set `localStorage.setItem('nvidia_api_key', 'nvapi-YOUR_KEY')`
3. Or pass through backend proxy

### API Rate Limit

**Problem:** "NVIDIA API Error: 429"

**Solution:**
- Free tier: 10 requests/min
- Upgrade plan at https://build.nvidia.com/
- Implement request debouncing
- Cache results in database

### Slow API Response

**Problem:** Analysis takes >5 seconds

**Solution:**
- Normal: NVIDIA API takes 2-4 seconds
- Check network speed: `curl https://integrate.api.nvidia.com`
- If consistently slow, use caching:

```javascript
// Cache API responses
const cache = new Map();

const analyzeGapCached = async (skills) => {
  const key = skills.sort().join('|');
  if (cache.has(key)) return cache.get(key);
  
  const result = await analyzeGap(skills);
  cache.set(key, result);
  return result;
};
```

### CORS Issues

**Problem:** "No 'Access-Control-Allow-Origin' header"

**Solution:** 
Use backend proxy (required for browser security):

```javascript
// Frontend
const response = await fetch('/api/analyze', {
  method: 'POST',
  body: JSON.stringify({ userSkills, apiKey })
});

// Backend handles NVIDIA API call
```

```javascript
const handleAnalysisComplete = async (results) => {
```

