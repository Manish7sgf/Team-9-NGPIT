# 🚀 NVIDIA AI Integration - Quick Reference

## In 30 Seconds

```jsx
// 1. Get API key from https://build.nvidia.com/api-keys

// 2. Create .env file
VITE_API_KEY=nvapi-YOUR_KEY_HERE

// 3. Use in your app
<SkillGapRadar 
  userSkills={['Java', 'Python']}
  enableAI={true}
/>

// 4. See AI-generated learning paths automatically!
```

---

## API Key Setup

### Option 1: Environment Variables (Recommended)
```bash
# Create .env in project root
VITE_API_KEY=nvapi-YOUR_KEY_HERE
VITE_ENABLE_AI_RECOMMENDATIONS=true

# Add to .gitignore
echo ".env" >> .gitignore
```

### Option 2: localStorage (For Testing)
```javascript
localStorage.setItem('nvidia_api_key', 'nvapi-YOUR_KEY');
```

### Option 3: Backend Proxy (Most Secure)
```javascript
// Frontend
fetch('/api/analyze', { body: JSON.stringify({ userSkills }) })

// Backend handles API with env var
```

## What Happens When User Analyzes

```
1. User enters skills: "Java", "Python", "SQL"
2. Clicks "Analyse Gap →"
3. Standard analysis:
   ✓ Matches against 7 future skills
   ✓ Calculates gap score
   ✓ Generates recommendations
4. AI analysis (if enableAI=true):
   ✓ Calls NVIDIA Llama 2
   ✓ Generates personalized paths
   ✓ Estimates time for learning
5. Results displayed with animations
   ✓ Shows all cards
   ✓ Shows AI learning paths
   ✓ Shows AI insights
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "No API key found" | Create `.env` with `VITE_API_KEY=nvapi-...` |
| 404 error | Check API key is correct |
| Slow (>5 sec) | Normal (2-4s), check network |
| Feature doesn't show | Check `enableAI={true}` |
| Want to disable AI | Use `enableAI={false}` |

---

## API Limits

- **Free Tier:** 10 requests/min
- **Response Time:** 2-4 seconds
- **Timeout:** 30 seconds
- **Fallback:** Uses default paths if unavailable
